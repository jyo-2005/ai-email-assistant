# 📧 MailMind AI

## 🚀 Overview
MailMind AI is a real-world AI email assistant that classifies emails and suggests actions like reply or ignore.

## 🎯 Features
- Spam detection
- Action recommendation
- Real-time UI
- API endpoints
- Baseline agent

## 🧠 Tasks
- Easy: classify email
- Medium: classify + action
- Hard: generate reply

## ⚙️ Setup

```bash
pip install -r requirements.txt
python -m uvicorn api:app --reload
